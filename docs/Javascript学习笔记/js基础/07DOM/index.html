07DOM

DOM（document object model）---js中通过DOM来对html文档进行操作

节点：网页中最基本的组成部分，每个部分都是节点

常用节点：文档--整个网页，元素---标签，属性---元素的属性，文本---标签中的文本内容

事件：文档和浏览器窗口发生一些特定的交互瞬间

处理事件:我们可以在事件对应的属性中设置一些代码，当事件触发时，这些代码将会执行

1.标签里绑定属性响应

2.绑定处理函数形式响应事件，事件触发时，函数被调用

btn.onclick = function(){
     alert("你点我干嘛");
}

浏览器加载页面时自上向下，如果将script标签写到head里，在代码执行时，页面还没有加载

如果要写到head里，可为页面加载完成事件绑定一个函数

onload：此事件会在整个页面加载完成后才触发

为window绑定onload

    window.onload = function(){
            //获取button对象
            var btn = document.getElementById("btn");
            console.log(btn);
            //修改按钮的文字
            btn.innerHTML = "我不是按钮";

            //处理事件
            //以函数形式
            btn.onclick = function(){
                alert("你点我干嘛");
            }
        }

写下边更优，差别不大

DOM里的查询方法

获取元素节点：

通过document对象调用

  1. getElementByTd()---通过id属性获取一个元素节点对象

  2. getElementsByTagName()--通过标签名获取一组元素节点对象

    返回类数组对象

  3. getElementsByName()---通过name属性获取一组元素节点对象

    返回类数组对象

innerHTML---通过这个属性可以获取到元素内部的html代码，对于自结束标签这个属性没有意义

如果需要读取元素节点的属性,直接使用元素.属性名，注：class属性不能采用这种方法，读取class属性时需要采用元素.className

获取元素节点的子节点：

通过具体的元素名调用

  1. getElementsByTagName()---返回当前节点的指定标签名后代节点

  2. childNodes---表示当前节点的所有子节点

    这个属性会获取包括文本节点（标签间的换行空白）在内的子节点，但是在IE8及以下浏览器中不会

    用children这个属性代替---可以获取当前元素的所有子元素

  3. firstChild---表示当前节点的第一个子节点（包括文本节点）

    firstElementChild获取当前元素的子元素（不包括文本节点），但是不支持IE8及以下

  4. lastChild---表示当前节点的最后一个子节点，同上

获取父节点和兄弟节点

通过具体的节点调用

  1. parentNode--当前节点的父节点

  2. previousSibling---前一个兄弟节点（可能获取到空白文本）

    previousElementSibling--前一个兄弟元素

  3. nextSibling---下一个兄弟元素

定义函数专门用来指定元素绑定单机响应函数

innnerText---和innerHtml类似，不同的是该元素可以获取到元素内部的文本内容，自动将html标签去除

DOM剩余查询方法

1.获取body---在document里有一个属性body，他保存的是body的引用

2.document.documentElement---保存的是html标签

3.document.all---页面中所有的元素 和document.getElementsByTagName（”*“）效果相

4.根据元素的class属性查询一组元素节点：

document.getElementsByClassName---IE8及以下不支持

5.document.querySelector()---需要一个选择器的字符串作为参数，根据一个css选择器查询一个元素节点对象，但是使用该方法总会返回唯一的元素，如果满足条件的元素有多个，只会返回第一个

6.document.querySelectorAll()---返回数组，可选择所有元素

DOM的操作

1.操作html结构

一。创建新节点：

1.创建元素节点：document.createElement（）；---需要一个标签名作为参数

2.创建文本节点：document.createTextNode（）；---文本内容作为参数

3.将文本节点添加到元素节点里：appendChild（）---把新的字节点添加到指定节点

语法：父节点.appendChild（子节点）

4.将新节点添加到父节点里，方法同3

通过更改innerHTML属性：父节点.innerHTML += 新节点.innerHTML，但是这种方法会将已有的子节点也重新换成新的，若那些节点已经绑定事件，起不了作用

我们一般结合使用：

创建新节点，设置innerHTML属性值，添加到父节点里

二。将指定节点插入到指定位置

1.将新节点插到一个节点前面：insertBefore（）---在指定子节点前面插入新的子节点

语法：父节点.insertBefore（新节点，旧节点）

2.替换指定节点：replaceChild（）---替换子节点

语法：父节点.replaceChild（新节点，旧节点）

三。删除节点

removeChild（）---删除字节点 语法：父节点.removeChild（子节点）或子节点.parentNode.removeChild（子节点）
