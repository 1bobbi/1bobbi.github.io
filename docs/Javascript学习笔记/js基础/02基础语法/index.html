02基础语法

JS基础

​ 输出语句：

​ 控制浏览器输出警告框---alert（“”）

​ 让计算机在页面中输出内容---document.write("")

​ 向控制台输出一个内容---console.log("")

<script>
       alert("hhh");
       document.write("hhh");
       console.log("hhh");
</script>

​ 编写位置：

​ 1.script标签中

​ 2.标签属性中：

​ 标签的onlick属性中 

    <button onlick="alert('hahah');">点一下</button>
    <!--
        当点击按钮时，js代码才会执行
    -->

​ 写在超链接的href属性中，点击超链接时才会执行

        <a href="javascript:alert('hahah');">超链接</a>

​ 写在标签属性中，属于结构与行为耦合，不方便维护

​ 3.写在外部js文件中，通过script标签引入

        <script src="JS/script.js"></script>

​ 一个script标签一旦用于引入外部文件，就不能编写内部代码，需要重新创建

​ 基本语法：

​ 注释：/* */多行注释 //单行注释

​ js中严格区分大小写，每条语句以分号结尾，会忽略多个空格和换行

​ 字面量和变量：

​ 字面量：不可改变的值，可以直接使用

​ 变量：保存字面量，通过使用var声明一个变量

        var a;
        a = 1;
        console.log("a");

​ 标识符：在js中所有可以由我们自主命名的都可称为标识符，如：变量名，函数名

​ 命名规则：

​ 1.标识符可含字母，数字，下划线，$

​ 2.标识符不能以数字开头

​ 3.不能用ES中的关键字或保留字

​ 4.采用驼峰命名---首字母小写，每个单词开头字母大写

​ js保存标识符采用的是UTF-8编码

​ 数据类型：指字面量的类型

​ String---字符串 Number---数值 Boolean---布尔值 Null---空值

​ Undefined---未定义 Object---对象（引用数据类型）

​ 基本数据类型：

​ 1.String

​ --在js中字符串需要使用引号（双或单）引起来

​ --引号不可嵌套，双里不用双，单里不用单，在字符串中可使用\作为转义字符

                var str = "hello";
                str = '他说：“今天天气真不错”';
                str = "他说：\“今天天气真不错\”";

​ 2.Number

​ --js中所有数值都是Number类型

​ --可使用typeof检查变量类型，检查数值时会返回Number

                    var a = 123;
                    var b = "123";
                    console.log(typeof a);
                    console.log(typeof b);

​ --数值最大值：Number.MAX-VALUE，使用Number超过最大值，会返回Infinity，Infinity是Number类型，最小值返回5e-324，大于0的最小值

​ --NaN是一个特殊的数字，表示Not a Numebr，类型也是Number

​ --计算：在js里整数的运算基本可以保证精确

​ 3.Boolean

​ --只有两个（true，false），逻辑判断

​ 4.Null，Undefined

​ --Null和Undefined类型只有一个值，就是null和undefined。null专门用来表示空对象，使用typeof检查时返回object，undefined指声明了未赋值

​ 强制转换：

​ 1.转换为String

​ 方式一：调用被转换数据类型的toString（）方法，该方法不会影响到原值，返回值会改变

    var a = 123;
    a = a.toString();

​ null和undefined没有此方法，调用会报错

​ 方式二：调用String（）函数，并将被转换的作为参数

    var a = 123;
    a = String(a);

​ null和undefined可调用

​ 2.转换为Number

​ 方式一：使用Number（）函数

​ 字符串-->数字

​ 1.若是纯数字字符串，则直接将其转换为数字

​ 2.若含非数字，则转换为NaN

​ 3.若为空串或全是空格，转为0

​ Boolean-->数字

​ true转为1，false转为0

​ Null-->数字 0

​ Undefined-->数字 NaN

​ 方式二：专用于字符串。非String使用会先转为String再操作

​ parseInt（），将字符串中有效的整数取出来转为整数；

​ parseFloat（）作用和上相同，获取有效小数

​ 其它进制数字：0x开头--十六进制 0开头--八进制 0b开头--二进制，但不是所以浏览器都支持

​ 进制转换---在parseInt（）中传递第二个参数，指定数字的进制

​ 3.转换为Boolean

​ 方法：使用Boolean（）函数

​ 数字--->布尔 除了0和NaN都是true

​ 字符串--->布尔 除了空串都是true

​ null和undefined false

​ 对象也会转为true

​ 运算符：

​ 1.算术运算符（+ - * / %）

​ +：

​ 对非Number进行计算时，会进行转换；（- * / %）

        var b = null + true;
       document.write(b);// 1

​ 任何值和NaN运算，结果都是NaN；

        var c = NaN + 1;
       document.write(c);//   NaN

​ 特例：若对字符串进行相加，则会拼串，要拼较长的字符串，可在每行添加加号；

​ 任何值和字符串做加法，都会先转换成字符串，并做拼串操作

        var a = "1" + 2;
       document.write(a);// 12
       var str = "fg" + "763";
       document.write(str);//   fg763

​ 2.一元运算符---只需要一个操作数

​ + 正号 对数值并不会产生影响

​ - 负号 符号取反

​ 对于非Number的值，会先转换为数值，再进行运算

​ 3.自增和自减（++ --）

​ 4.逻辑运算符

​ ！非 --- 对布尔值取反；对非布尔值取反，先转换为布尔值再取反

​ &&与 --- 布尔值：只要有一个false，就返回false；短路规则：若第一个值为false，则直接返回false

​ ||或 --- 布尔值：只要有一个为true，则返回true；短路规则：若第一个值为true，则直接返回true

​ 非布尔值：进行与或运算时，会先转换为布尔值，再运算，并返回原值

与运算：若第一个值为true，则返回第二个值；若第一个值为false，返回第一个值

或运算：若第一个值为true，则返回第一个值；若第一个值为false，返回第二个值

​ 5.赋值运算符（ = += -= *= /= %=）

​ 6.关系运算符(> >= < <=)

​ 是数值按数学规则比较；非数值进行比较时会将其转换为数字，再比较；任何值和NaN作比较，都是false

        document.write(1 > 2);//false
        document.write(true > false);//true
        document.write(1 > NaN);//false

​ 特例：若符号两侧值都是字符串，不会将其转换为数字比较，而回分别比较字符串中字符的Unicode编码，且是一位一位比较如果两位一样，则比较下一位，可用来对英文进行排序

        document.write("aasd" > "fghj");//false

​ Unicode编码---UTF-8编码 输出Unicode编码---“\u+编码” 在网页中使用---“&#+编码+；”，且在这里需要的编码是十进制

​ 7.相等运算符（==）和不等运算符（！=）

​ 相等：若值的类型不同，则会自动将其转换为相同的类型，再比较；undefined衍生自null，两个值相等；NaN不和任何值相等，包括它本身——判断一个值是否是NaN，通过isNaN（）函数

            document.write("dfg" == 45);//false
            document.write(undefined == null);//true
            document.write(1 == NaN);//false
            document.write(NaN == NaN);//false

​ 不等比较规则和相等相同

​ ===（全等）--- 用来判断两个值是否全等，和相等类似，不同的是不会进行自动转换 注：null和undefined相等但不全等

​ ！==（不全等） 和全等类似

​ 若为非布尔值，先转换

​ 9. “，”---可分割多个语句，声明多个变量并赋值

​ 代码块---在js中可使用{}来为语句进行分组，代码块内部内容在外部也可见
