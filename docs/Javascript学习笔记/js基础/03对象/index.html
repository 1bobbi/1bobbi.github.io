03对象

对象---object，引用数据类型

基本数据类型都是单一的值，值和值之间没有联系

对象属于复合的数据类型，在对象中可以保存多个不同的数据类型

对象的分类：

  1. 内建对象 由ES标准定义的对象，在任何ES的实现中都可以使用 如：Math String
  2. 宿主对象 由JS运行环境提供的对象，目前来讲主要指浏览器提供的对象 如：BOM DOM
  3. 自定义对象

1.创建对象

var obj = new Object();
//使用new关键字调用的函数，是构造函数，构造函数是专门来创建对象的函数

使用工厂方法创建对象：

    function creat(a,b,c){
            //创建新对象
            var obj = new Object();
            //添加属性
            obj.name = a;
            obj.age = b;
            obj.genger = c;
            return obj;
        }
        var obj2 = creat();

使用这种模式使用的构造函数都是object，所以创建的对象都是object类型，无法区分不同的对象

2.添加属性

语法：对象.属性名 = 属性值

obj.name = "xiaoming";

3.读取属性

语法：对象.属性名 读取对象中没有的属性不会报错，会返回undefined

4.修改属性值---和添加语法相同

5.删除属性

语法：delete 对象.属性名

属性名：对象属性名不强制要求遵守标识符规范，如果要使用特殊属性名，语法：对象["属性名"] = 属性值，读取时也要采取这种方式

obj["123"] = 456;
console.log(obj["123"]);

可以使用[ ]这种形式去操作属性，更加灵活

var n = "123";
console.log(obj[n]);

属性值：可以是任意的数据类型

var obj2 = new Object();
obj2.name = "xiaohong";
obj.test = obj2;

方法：

//属性值也可以是函数
  obj.name = function(){

   }
 //一个函数作为一个对象的属性保存，则称这个函数是这个对象的方法，调用函数就是调用对象的方法

枚举对象中的属性：

使用for……in语句

语法：for（var 变量 in 对象）{

​ 语句

}

有几个属性，语句就会执行几次，每次执行时会将一个属性赋值给变量

for(var n in obj){
     console.log(n);
     console.log(obj[n]);
 }

in 运算符：可检查对象中是否含有指定属性

语法：“属性名” in 对象

基本数据类型与引用数据类型

js中基本数据类型直接保存在栈内存中，值与值之间独立

对象保存在堆内存中，每创建一个新的对象就会在堆内存中开辟一个新的空间，而变量保存到是对象的内存地址

        var a = 10;
        var b = a;
        a++;
        var o1 = new Object();
        o1.name = "hh";
        var o2 = o1;
        o2 = null;//只代表o2和o1的联系断开了，不能改变o1的值

比较两个基本数据类型比较的是值，比较两个引用数据类型比较的是内存地址

        var o3 = new Object();
        var o4 = new Object();
        o3.name = "aa";
        o4.name = "aa";
        console.log(o3 == o4);//false

对象字面量

使用对象字面量创建对象，在创建对象时可直接指定对象中的属性---语法：{属性名：属性值，属性名：属性值……}

var obj = {};
obj.name = "b";
var obj = {name:a,age:12};

Date对象---表示一个时间

如果直接使用构造函数创建Date对象，则会封装为代码执行的时间

var d = new Date（）；

创建指定的时间对象需要在构造函数中传递字符串作为指定参数

var d2 = new Date（4/04/2022 11:23:12）；月份/日/年 时/分/秒

Date的方法：

getDate( ) --- 获取当前日期对象是几日

getDay( ) --- 获取当前日期对象是周几（0-6）0是周日

getMonth( ) --- 获取当前日期对象月份（0-11）0是1月

getFullYear( ) --- 获取当前日期对象年份

getTime( ) --- 获取当前日期对象的时间戳，时间戳指从1970年1月1日，0时0分0秒到当前日期的毫秒数（1秒=1000毫秒），计算机底层保存时间时会使用

now( ) --- 获取当前时间戳

Math对象---不是构造函数，属于工具类，不用创建对象，他里边封装了数学运算相关属性和方法

abs（）--- 返回数的绝对值

ceil（）--- 对数进行上舍入-向上取整

floor（）--- 向下取整

round（）--- 四舍五入

random（）--- 生成0-1之间的随机数 random（）*10--- 生成0-10之间的 round（random（））---可取到端点

round（random（）*（y-x）+x）--- 生成任意两个数之间

max（）--- 多个数中的最大值 min（）反之

pow（x，y）--- x的y次幂 sqrt（x）--- 开x方根
