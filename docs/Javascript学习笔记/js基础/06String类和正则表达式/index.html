06String类和正则表达式

包装类

对象之间按地址比较

在js中有三个包装类，通过这三个包装类可以将基本数据类型转换为对象

String（）---将基本数据类型字符串转换为String对象

Number（）---将基本数据类型数字转换为Number对象

Boolean（）---将基本数据类型布尔值转换为Boolean对象

方法和属性只能添加给对象，当对基本数据类型调用属性和方法时，浏览器会自动将其转换为对象，然后调用对象的属性和方法

String对象的方法：

在底层字符串以字符数组保存，所以字符串和数组的操作方法类似

length---用来获取字符串的长度

charAt（x）--- 返回指定位置的字符

charCodeAt（）--- 返回指定位置字符的unicode编码

fromcharCode（）--- 根据字符编码获取字符

语法：构造函数.fromcharCode（）

concat（）--- 连接两个或多个字符串，作用和+一样

indexOf（）--- 检索字符串中是否含有指定内容，含有则返回其第一次出现的索引，没有则返回-1，可以指定第二个参数指定开始查找的位置

lastindexOf（）--- 用法和indexOf（）一样，不同的是该方法是从后往前找

slice（）--- 截取指定内容,[参数1，参数2) -1表示倒数第一个

substring( ) --- 和slice（）类似，不同的是这个方法不能接收负数，若传递了，默认为0，而且它会自动调整参数位置，若第二个参数小于第一个自动交换

substr（）--- 作用和slice（）相同，不同的是第二个参数表示截取数量

split（）--- 将字符串拆分为数组，有实际参数根据参数拆分，传递空串每个元素都被拆分

toLowerCase（）--- 将字符串转换为小写 toUpperCase（）相反

和正则相关的方法：

split（）在方法中传递正则表达式作为参数，会根据任意字母将字符串拆分

即使不指定全局也会全部拆分

split（/[ a - z ]/)

search（）--- 搜索是否有指定内容

只会查找第一个，即使设置全局匹配也没用

search（/a[bde]c/)

match（）--- 根据正则表达式从字符串中将符号条件的内容提取出来

默认情况下找到第一个符合条件的就停止，可设置全局模式

match（/[a - z]/g)

可谓正则表达式设置多个匹配模式

match会将匹配到的内容封装到数组中返回

replace（参数1，参数2）--- 将字符串中的指定内容替换为新的

1是被替换的，2是新的

默认只替换第一个，可设置全局模式

replace（/[a - z]/g," ")

正则表达式

用于定义一些字符串的规则，计算机可以根据正则表达式检查字符串，或将字符串中符合规则的内容提取出来

1.创建正则表达式对象

语法： var 变量 = new RegExp(“正则表达式”，“匹配模式”);

var reg = new RegExp("a");

使用字面量创建

语法：var 变量 = /正则表达式/匹配模式

var reg = /a/i;

2.检查

方法：test（）--- 检查字符串中是否含有正则表达式对象参数，严格区分大小写

var str = "a";
reg.test(str);
reg.test("asdf");

匹配模式可选值：

i 忽略大小写

g 全局匹配

var reg = new RegExp("a"，"i");
var str = "a";
reg.test("Asdf");

*检查是否有a或b或c……

使用|表示或

var reg = /a|b/i;
reg.test("Asdf");

*检查是否有字母

[ ]里的内容也是或的关系

[ a - z ]表示任意小写字母

var reg = /[a - z]/i;
reg.test("b");

*检查字符串中是否有abc或adc或aec

var reg = /a[bde]c/i;
reg.test("abc");

[ ^ab ]表示除了ab以外

量词：通过量词可以设置一个内容出现的次数，量词只对他前边的一个内容起作用

  • { n } 正好出现n次
  • { x ，y } 出现x - y 次
  • {x ，} x次以上
  • “ + ” 至少一个
  • “ * ” 0个或多个
  • “ ？” 0个或一个

*检查字符串是否以a开头

^ 表示开头 $ 表示结尾

reg = /^a/;
reg = /a$/;
reg = /^a$/;//只有一个a     abca

*检查字符串中是否有" . "

“ . ” 表示任意字符，所以要使用转义字符

reg = /\./;
reg = /\\/;检查"\"

特殊符号：

\w---任意字母，数字，下划线 [0-9A-z] \W---除了字母，数字，下划线 加^

\d---数字 \D---除了数字

\s---空格 \S---除了空格

\b--单词边界 \B---除了单词边界

检查是否有单词child

reg = /child/;
console.log(reg.test("hello children"));//true
reg = /\bchild\b/;//false正确写法

去除字符串中的空格---使用空串替代空格

var str = "         hello           ";
str = str.replace(/\s/g,"");//中间的空格也会一起去除
str = str.replace(/^\s*/g,"");//开头空格
str = str.replace(/\s*$/g,"");//结尾
str = str.replace(/^\s*|\s*$/g,"");//开头和结尾
