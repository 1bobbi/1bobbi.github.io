01基础总结深入

基础总结

一，数据类型

1.分类

基本（值）：

String：任意字符串 Number：任意数字 Boolean：true false Undefined：undefined

null：null

对象（引用）：

object：任意对象

function：特殊的对象（可执行）

array：特殊的对象（数值下标，内部数据有序）

2.判断：

typeof：返回数据类型的字符串表达，返回的是小写，并且判断null返回的是object

​ 可以判断：undefined / 数值 / 字符串 / 布尔值 / function

​ 不能判断：null和object，object和array

instanceof：判断对象的具体类型，返回布尔值

===

​ 可以判断：undefined

3.相关问题

undefined和null的区别？

undefined代表定义了未赋值

null代表定义并赋值了，赋值为null

什么时候给变量赋值为null？

初始赋值，表明变量将要赋值为对象

结束前，让对象成为垃圾对象，被垃圾回收器回收

严格区别变量类型与数据类型？

数据的类型：

基本类型和对象类型

变量的类型（变量内存值的类型）： 基本类型（保存基本类型的数据）和引用类型（保存的是地址值）

二，数据，变量和内存

1.什么是数据？

存储在内存中的代表特定信息的东西，本质是0101……

数据的特点：可传递，可运算

一切皆数据内存中所有操作的目标：数据

​ 算数运算，逻辑运算，赋值，运行函数

2.什么是内存？

内存条通电以后产生的可存储数据的空间（临时的）

内存产生和死亡：内存条（电路板）==》通电==》产生内存空间==》存储数据==》处理数据==》断电==》内存空间和数据消失

一块小内存的两个数据：内部存储数据，地址值数据

内存分类：

栈：全局变量和局部变量

堆：对象

3.什么是变量？

可变化的量，由变量名和变量值组成

每个变量都对应一块小内存，变量名用来查找对应的内存，变量值就是内存中保存的数据

4.内存，数据，变量三者之间的关系？

内存是用来存储数据的空间，变量是内存的标识

5.相关问题

var a = xxx, a内存中保存的是什么？

​ xxx是基本数据，保存的就是这个数据

​ xxx是对象，保存的是对象的地址值

​ xxx是变量，保存的是xxx的内存内容（基本数据，地址值）

引用变量问题

var obj1 = { name：'tom'};
var obj2 = obj1;
obj1.name = 'jack';
console.log(obj2.name);  // jack

多个引用变量指向同一个对象，通过其中一个变量修改对象内部的数据，其他所有变量看到的是修改后的数据

2个引用变量指向同一个对象，让其中一个变量指向另一个对象，另一变量依然指向前一个对象

var a = {age:12};
fuction fn(obj){// obj = {age:12}
    obj = {age:15};
}
fn(a);
console.log(a.age)// 12

在js调用函数时传递变量参数时，是值传递还是引用传递？

有两种理解，可以都是值传递（基本，地址值），也可以是值传递和引用传递（地址值）

js引擎如何管理内存？

内存生命周期：分配小内存空间，得到她的使用权；存储数据，可以进行反复操作；释放小内存空间

释放内存：局部变量：函数执行完自动释放；对象：成为垃圾对象==》垃圾回收器回收

三，对象

1.什么是对象？

多个数据的封装体或是用来保存多个数据的容器

一个对象代表现实中的一个事物

2.为什么要使用对象？

统一管理多个数据

3.对象的组成？

属性（属性名和属性值），方法（一种特别的属性==》属性值是函数）

4.如何访问对象的内部数据？

" . " + 属性名（编码简单，但有时不能用）；[ ' 属性名 ' ]（通用）

5.什么时候必须用[ ' 属性名 ' ]的方式？ 属性名包含特殊字符：“ - ”，空格

​ 属性名不确定

var propName = 'myAge';
var value = 18;
//p.propName = value; 不可用
p[propName] = value;

四，函数

1.什么是函数？

实现特定功能的n调语句的封装体

只有函数是可以执行的，其他类型的数据不能执行

2.为什么要使用函数？

提高代码复用；便于阅读交流

3.如何定义函数？

函数声明

function fn1(){
    console.log('fn1');
}

表达式

var fn2 = function(){
    console.log('fn2')
}

4.如何调用函数？

直接调用；通过对象调用；new调用；

test.call/apply ( obj ) : 临时让test成为obj的方法进行调用

var obj = {};
function test(){
    this.xxx = 'hello';
}
//obj.test(); 不能直接调用
test.call(obj); // 可以让一个函数成为指定任意对象的方法进行调用
console.log(obj.xxx);

5.回调函数

5.1什么函数才是回调函数？

自定义，自己没有调用，但最终他执行了（在某个时刻或某个条件下）

document.getElementById("btn").onclick = function(){// dom事件回调函数
    alert(this.innerHTML);
}
//扩展：
/*定时器：超时定时器==》setTimeout()  可以在规定时间（以毫秒计）过后执行一次代码块；
循环（延时）定时器==》setInterval()  按照指定的周期（以毫秒计）来重复执行某些代码。*/
setTimeout(function(){//定时器回调函数
    alert("hello");
},2000)

5.2常见的回调函数？

dom事件回调函数；定时器回调函数；ajax回调函数；生命周期回调函数

六，IIFE

1.理解

全称： Immediately-Invoked Function Expression --->立即执行函数

2.作用

隐藏实现；不会污染外部（全局）命名空间；

可以用他来编码js模块

(function(){
    var a = 1;
    function test(){
        console.log(++a);
    }
    window.$ = function(){// 相当于给window添加了一个叫做$的属性，属性值为函数
        return {
            test:test //相当于返回了一个对象，这个对象叫做test，他的值为上面的test函数
        }
    }
})()
$().test();

七，this

1.this是什么？

任何函数本质上都是通过某个对象来调用的，所有函数内部都有一个变量this，它的值是调用函数的当前对象

如果没有指定就是window

2.如何确定this的值？

​ test( ) : window

​ p.test( ) : p

​ new test( ) : 新创建的对象

​ p.call( obj ) : obj

function Persion(color){
    console.log(this);
    this.color = color;
    this.getColor = function(color){
        console.log(this);
        return this.color;
    }；
    this.setColor = function(color){
        console.log(this);
        return this.color;
    }；
}

Persion("red");// this是谁？ 
var p = new Persion("yellow"); //this是谁？
p.getColor();//this是谁？
var obj = {};
p.setColor.call(obj,"black");//this是谁？
var test = p.setColor;
test();//this是谁？
function f1(){
    function f2(){
        console.log(this);
    }
    f2();//this是谁？
}
f1();
